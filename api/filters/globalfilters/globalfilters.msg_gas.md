# GlobalFilters.MSG\_GAS

The MSG\_GAS property is used to add a filter to include or exclude functions that calls `msg.gas`.&#x20;

### Query Example

```python
from glider import *


def query():
  functions = (
    Functions()
    .with_globals(GlobalFilters.MSG_GAS)
    .exec(1)
  )
  
  return functions
```
