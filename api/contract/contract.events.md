# Contract.events()

`events() â†’ List[Event]`

## Example query

```python
from glider import *

def query():
  contracts = Contracts().exec(5)
  
  names = []
  for contract in contracts:
    events = contract.events()

    for event in events:
      names.append(event.name)

  return [{"names": names}]
```
