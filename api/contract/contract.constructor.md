# Contract.constructor()

## Example query

```python
from glider import *

def query():
  contracts = Contracts().exec(10)
  
  constructors = []
  for contract in contracts:
    constructor = contract.constructor()

    if not isinstance(constructor, NoneObject):
      constructors.append(constructor)

  return constructors
```
