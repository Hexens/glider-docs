# Contract.source\_code()

## Return type

str

## Example query

```python
from glider import *

def query():
  contracts = Contracts().exec(2)
  
  codes = []
  for contract in contracts:
    code = contract.source_code()
    codes.append(code)

  return [{"codes": codes}]
```
