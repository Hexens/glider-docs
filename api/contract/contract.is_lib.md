# Contract.is\_lib()

## Return type

bool

## Example query

```python
from glider import *

def query():
  contracts = Contracts().exec(25)
  
  libs = []
  for contract in contracts:
    if contract.is_lib():
      libs.append(contract)

  return libs
```
