# Contract.address()

## Return type

str (e.g. `"0x798AcB51D8FBc97328835eE2027047a8B54533AD"`)

## Example query

```python
from glider import *

def query():
  contracts = Contracts().with_name("ERC721").exec(5)
  
  addresses = []
  for contract in contracts:
    addresses.append(contract.address())

  return [{"addresses": addresses}]
```
